--Password Manager
This is a simple password manager GUI application built with Python and Tkinter. The program allows users to generate random passwords, save their login credentials for different websites, and retrieve them as needed.

--Features
Password generator to create strong and secure passwords
Save login credentials (website, username, password) to a local file
Copy password to clipboard with one click

--Requirements
Python 3.0 or later
Tkinter (included in most Python distributions)
pyperclip module (for copying passwords to clipboard)

--How to use
Run the program using the command python password_manager.py in your terminal.
Enter the website, username, and password in their respective fields.
Click the "Add" button to save the login credentials to a file.
To generate a password, click the "Generate password" button.
To copy the password to clipboard, click the password field.
